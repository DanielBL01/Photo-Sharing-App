Website functionalities
  • The homepage will show the latest 25 posts made by any user at one time
  • A user is able to filter and search a post using keywords to match titles to any number of posts made by any user
  • A user is given the option to comment any number of times on any given post, including their own posts
  • If a staff user comments, it will be indicated with (admin)
  • Only the user who created the post will see options to either delete their post along with all the comments regarding that post or update the post's title and/or description
  • A user is also allowed to update the title and/or description any number of times
  • To signup, I have only made the username, password, and comfirmation password manitory so users do not need to give information about their email, name, age or bio
  • The timezone for this website is set to America/Toronto. Click here to see the list of valid timezones
